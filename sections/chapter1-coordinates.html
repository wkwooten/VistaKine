1.2 Coordinate Systems

Cartesian Coordinates

Cartesian coordinates use three perpendicular axes (X, Y, Z) to represent position in 3D space.

A point P is represented by three values (x, y, z) that indicate the distance along each axis.

Spherical Coordinates

Spherical coordinates represent points using:

  • r: Distance from the origin (radius)
  • θ (theta): Polar angle (from the positive z-axis)
  • φ (phi): Azimuthal angle (in the xy-plane from the x-axis)

This system is particularly useful for problems with spherical symmetry.
