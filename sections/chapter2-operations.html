2.2 Vector Operations

Learn how to perform basic operations with vectors.

Vector Addition and Subtraction

Vectors can be added by combining their components:

\( \vec{a} + \vec{b} = (a_x + b_x, a_y + b_y, a_z + b_z) \)

Geometrically, this is represented by the parallelogram rule or tip-to-tail method.

In the visualization, drag the vectors to see how their sum changes. Notice that vector addition is commutative: \( \vec{a} + \vec{b} = \vec{b} + \vec{a} \).

Scalar Multiplication

Multiplying a vector by a scalar changes its magnitude while preserving its direction (or reversing it if the scalar is negative):

\( c\vec{v} = (cv_x, cv_y, cv_z) \)

When c > 1, the vector is stretched. When 0 < c < 1, the vector is shortened.

When c < 0, the vector points in the opposite direction.
